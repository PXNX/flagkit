# flagkit
Flag Kit copy from https://github.com/madebybowtie/FlagKit for Android 

**build.gradle**
    
    implementation 'com.jwang123.flagkit:flagkit:1.0'

**Usage**

    FlagKit.drawableWithFlag(context, "Country code");
   
   
**Example**
   
    FlagKit.drawableWithFlag(context, "US");
