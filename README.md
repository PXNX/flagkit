# flagkit
Flag Kit copy from https://github.com/madebybowtie/FlagKit for Android 

Usage
    
    compile 'com.jwang123.flagkit:flagkit:1.0'
    
    FlagKit.drawableWithFlag(context, "Conutry code");
    
    FlagKit.drawableWithFlag(context, "US");
